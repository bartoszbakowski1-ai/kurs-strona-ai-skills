# Brief LP - przykład do skilla `/lp`

Cel strony: zapis na live / replay "Stwórz stronę z AI".

Typ LP: webinar/live, później replay.

Grupa docelowa: osoby, które już używają Claude albo ChatGPT, ale nie używały Claude Code i chcą zbudować własną stronę bez uczenia się programowania od zera.

Obietnica: pokażemy, jak przejść z samego czatu do systemu, który realnie buduje stronę, formularz, landing page, ofertę i publikację.

CTA: "Chcę dostęp"

Termin: 17 czerwca 2026 wieczorem. Godzina: `[do potwierdzenia]`.

Pierwszy trust signal: prowadzący pokazuje realny system pracy z Claude Code: Notion + GitHub Course Kit + skille + dokumenty strategiczne.

Co osoba dostaje po zapisie lub zakupie:

- replay live'a,
- dostęp do kursu w Notion,
- GitHub Course Kit do Claude Code,
- skille, prompty i karty strategiczne,
- wideo lekcje dodane później bez dopłaty.

Sekcje:

1. Hero z obietnicą, terminem i CTA above the fold.
2. Dla kogo jest live.
3. Co zbudujesz po live.
4. Co jest w pakiecie.
5. Dlaczego to nie jest zwykły kurs tekstowy.
6. Jak działa Notion + GitHub + Claude Code.
7. FAQ.
8. Finalne CTA.

Oferta: 397 zł do niedzieli dla osób z live'a. Później cena rośnie.

Ważne ograniczenie: nie obiecuj, że każdy bez wysiłku zrobi idealną stronę. Obietnica ma brzmieć: dostajesz system pracy, który prowadzi Cię przez proces i ogranicza chaos.

Rzeczy do oznaczenia, jeśli nie są dostępne w projekcie:

- godzina live'a,
- docelowy link formularza,
- realne zdjęcie prowadzącego,
- screen panelu Notion,
- screen repo GitHub.
