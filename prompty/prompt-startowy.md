# Prompt startowy

Użyj go po zainstalowaniu paczki kursowej w projekcie strony.

```plain text
Zaczynam kurs "Stwórz stronę z AI".

Najpierw przeczytaj:
- .claude/materialy/kurs/M0.md
- .claude/materialy/prompty/prompt-ratunkowy.md
- README.md, jeśli istnieje w projekcie

Nie buduj jeszcze strony.
Sprawdź, czy mam zainstalowane skille kursowe w .claude/skills/ i komendę .claude/commands/strona.md.

Potem poprowadź mnie przez pierwszy krok: stworzenie mojego kontekstu osobistego i zawodowego.
```
