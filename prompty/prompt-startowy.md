# Prompt startowy

Uzyj go po zainstalowaniu paczki kursowej w projekcie strony.

```plain text
Zaczynam kurs "Stworz strone z AI".

Najpierw przeczytaj:
- .claude/materialy/kurs/M0.md
- .claude/materialy/prompty/prompt-ratunkowy.md
- README.md, jesli istnieje w projekcie

Nie buduj jeszcze strony.
Sprawdz, czy mam zainstalowane skille kursowe w .claude/skills/ i komende .claude/commands/strona.md.

Potem poprowadz mnie przez pierwszy krok: stworzenie mojego kontekstu osobistego i zawodowego.
```
