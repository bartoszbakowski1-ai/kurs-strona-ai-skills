# M4 - Design: zeby NIE wygladalo jak AI

## Cel

Ustawiasz kierunek wizualny i przygotowujesz media, zeby strona nie wygladala jak generyczny landing wygenerowany przez AI.

## Najczestszy AI-look

- fioletowo-niebieskie gradienty,
- za duzo zaokraglonych kart,
- puste ikony zamiast dowodow,
- brak realnych zdjec,
- hasla bez konkretu,
- symetryczne sekcje bez rytmu,
- wszystko wyglada poprawnie, ale bez charakteru.

## Krok 1: design

```plain text
Uruchom skill design.

Przeczytaj karty/karta-wizualna.md i zamien ja w spojny design system calej strony-systemu (wspolny dla wszystkich podstron): kolory, fonty, spacing, radius, zasady layoutu i reguly anti-ai-look. Nie buduj jeszcze sekcji. Przygotuj fundament wizualny.
```

## Krok 2: obrazy

```plain text
Uruchom skill obrazy.

Na podstawie mojej karty wizualnej i architektury tresci zaproponuj minimum 3 realne media do strony. Dla kazdego powiedz, gdzie ma sie pojawic i jaka funkcje pelni. Unikaj plastikowych ilustracji AI i pustych ikon.
```

## Krok 3: sprawdz braki

```plain text
Ocen kierunek wizualny przez anti-ai-look. Wskaz 5 najwazniejszych zmian, ktore sprawia, ze strona bedzie wygladac jak konkretna marka, a nie domyslny szablon.
```

## Zasada mediow

Jezeli nie masz zdjec, nie udawaj, ze masz. Zrob plan:

- co mozesz sfotografowac telefonem,
- jakie screeny mozesz pokazac,
- jakie mockupy mozna przygotowac,
- co tymczasowo moze byc placeholderem,
- co trzeba wymienic przed publikacja.

## Efekt koncowy M4

Masz kierunek wizualny, zasady anti-ai-look i plan mediow. Strona moze teraz powstawac bez wpadania w generyczny styl.
