# M4 - Design: żeby NIE wyglądało jak AI

## Cel

Ustawiasz kierunek wizualny i przygotowujesz media, żeby strona nie wyglądała jak generyczny landing wygenerowany przez AI.

## Najczęstszy AI-look

- fioletowo-niebieskie gradienty,
- za dużo zaokrąglonych kart,
- puste ikony zamiast dowodów,
- brak realnych zdjęć,
- hasła bez konkretu,
- symetryczne sekcje bez rytmu,
- wszystko statyczne i płaskie, nic nie ożywa przy przewijaniu,
- wszystko wygląda poprawnie, ale bez charakteru.

Drugą stroną medalu jest to, czego brakuje: ruch, głębia, kontrast typografii, jeden zapamiętywalny moment. Brak AI-tellów to jeszcze nie dobry design. Dlatego skill `design` nie tylko ustawia kolory i fonty, ale też zakłada warstwę ruchu (delikatne wjeżdżanie sekcji przy przewijaniu) - fundament, który skill budujący nałoży potem na całą stronę.

## Krok 1: design

```plain text
Uruchom skill design.

Przeczytaj karty/karta-wizualna.md i zamien ja w spojny design system calej strony-systemu (wspolny dla wszystkich podstron): kolory, fonty, spacing, radius, zasady layoutu, warstwe ruchu (motion) i reguly anti-ai-look. Nie buduj jeszcze sekcji. Przygotuj fundament wizualny.
```

## Krok 2: obrazy

```plain text
Uruchom skill obrazy.

Na podstawie mojej karty wizualnej i architektury tresci zaproponuj minimum 3 realne media do strony. Dla kazdego powiedz, gdzie ma sie pojawic i jaka funkcje pelni. Unikaj plastikowych ilustracji AI i pustych ikon.
```

## Krok 3: sprawdź braki

```plain text
Ocen kierunek wizualny przez anti-ai-look. Wskaz 5 najwazniejszych zmian, ktore sprawi, ze strona bedzie wygladac jak konkretna marka, a nie domyslny szablon.
```

## Zasada mediów

Jeżeli nie masz zdjęć, nie udawaj, że masz. Zrób plan:

- co możesz sfotografować telefonem,
- jakie screeny możesz pokazać,
- jakie mockupy można przygotować,
- co tymczasowo może być placeholderem,
- co trzeba wymienić przed publikacją.

## Efekt końcowy M4

Masz kierunek wizualny, warstwę ruchu w fundamencie, zasady anti-ai-look i plan mediów. Strona może teraz powstawać bez wpadania w generyczny styl - i od razu z życiem, nie jako płaski szablon.
