# M5 - Budowa strony + formularz

## Cel

Claude Code buduje strone sekcja po sekcji na podstawie Twojego kontekstu i trzech kart.

## Zanim zaczniesz

Sprawdz, czy masz:

- `kontekst/`,
- `karty/`,
- ustawiony design z M4,
- minimum 3 media albo plan ich dodania.

Prompt kontrolny:

```plain text
Sprawdz, czy projekt jest gotowy do budowy strony: kontekst, karty, design, media i status git. Jesli czegos brakuje, zatrzymaj sie i powiedz mi co uzupelnic.
```

## Uruchom budowe

```plain text
Uruchom skill zbuduj-strone.

Buduj strone na podstawie folderow kontekst/ i karty/.
Pracuj w malych etapach: jedna sekcja = jeden batch.
Po kazdym wiekszym etapie aktualizuj PROGRESS.md.
Nie pytaj mnie o decyzje techniczne. Pytaj tylko o tresci, oferte, dowody i media.
```

## Formularz

Gdy strona ma miec kontakt:

```plain text
Dodaj prosty formularz kontaktowy z walidacja, bezpiecznym endpointem i konfiguracja przez zmienne srodowiskowe. Nie wpisuj sekretow w kodzie. Przygotuj .env.local.example bez prawdziwych wartosci.
```

Typowe zmienne:

```plain text
RESEND_API_KEY=...
KONTAKT_TO=...
KONTAKT_FROM=...
```

## Kontrola jakosci

Po zbudowaniu strony:

```plain text
Uruchom skill sprawdz-kod. Sprawdz build, mobile, formularz, anti-ai-look, dostepnosc podstawowa i czy w kodzie nie ma sekretow.
```

## Efekt koncowy M5

Masz lokalnie dzialajaca strone z formularzem albo gotowym planem formularza. Przed publikacja przechodzisz przez brame jakosci.
