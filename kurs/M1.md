# M1 - Setup: aplikacja Claude + Twój kontekst

## Cel

Masz działający projekt strony, zainstalowaną paczkę kursową i folder `kontekst/` z plikami o Tobie.

## Co powinno być gotowe

- aplikacja Claude / Claude Code,
- folder projektu strony,
- projekt Next.js,
- paczka kursowa zainstalowana w `.claude/`,
- skill `kontekst` uruchomiony i zakończony.

## Krok 1: sprawdź folder projektu

W Claude Code wklej:

```plain text
Sprawdz, czy jestem w folderze projektu strony. Pokaz sciezke, package.json i aktualny status git. Jesli nie ma projektu Next.js, poprowadz mnie przez utworzenie go najprosciej jak sie da.
```

## Krok 2: zainstaluj paczkę kursową

Jeśli jeszcze tego nie zrobiłeś/zrobiłaś, wklej prompt z:

```plain text
prompty/prompt-instalacja.md
```

Po instalacji powinny istnieć:

```plain text
.claude/skills/
.claude/commands/strona.md
.claude/materialy/kurs/
.claude/materialy/prompty/
.claude/materialy/30-szablony-md/
```

## Krok 3: uruchom kontekst

```plain text
Uruchom skill kontekst.
```

Claude Code zada Ci pytania o:

- kim jesteś,
- komu pomagasz,
- jaką masz ofertę,
- jaki masz styl komunikacji,
- jakich klientów chcesz przyciągać,
- czego nie chcesz komunikować.

Efekt:

```plain text
kontekst/
  profil.md
  persona.md
  procesy.md
```

## Krok 4: sprawdź kontekst

```plain text
Przeczytaj folder kontekst/ i powiedz mi, czy ten kontekst wystarczy, zeby przejsc do M2 i M3. Jesli czegos brakuje, zadaj maksymalnie 5 pytan.
```

## Efekt końcowy M1

Claude Code zna Twój zawodowy i osobisty kontekst. Od teraz kolejne skille nie zgadują od zera.
