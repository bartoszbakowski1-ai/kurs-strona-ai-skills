# M1 - Setup: aplikacja Claude + Twoj kontekst

## Cel

Masz dzialajacy projekt strony, zainstalowana paczke kursowa i folder `kontekst/` z plikami o Tobie.

## Co powinno byc gotowe

- aplikacja Claude / Claude Code,
- folder projektu strony,
- projekt Next.js,
- paczka kursowa zainstalowana w `.claude/`,
- skill `kontekst` uruchomiony i zakonczony.

## Krok 1: sprawdz folder projektu

W Claude Code wklej:

```plain text
Sprawdz, czy jestem w folderze projektu strony. Pokaz sciezke, package.json i aktualny status git. Jesli nie ma projektu Next.js, poprowadz mnie przez utworzenie go najprosciej jak sie da.
```

## Krok 2: zainstaluj paczke kursowa

Jesli jeszcze tego nie zrobiles/zrobilas, wklej prompt z:

```plain text
prompty/prompt-instalacja.md
```

Po instalacji powinny istniec:

```plain text
.claude/skills/
.claude/commands/strona.md
.claude/materialy/kurs/
.claude/materialy/prompty/
.claude/materialy/30-szablony-md/
```

## Krok 3: uruchom kontekst

```plain text
Uruchom skill kontekst.
```

Claude Code zada Ci pytania o:

- kim jestes,
- komu pomagasz,
- jaka masz oferte,
- jaki masz styl komunikacji,
- jakich klientow chcesz przyciagac,
- czego nie chcesz komunikowac.

Efekt:

```plain text
kontekst/
  profil.md
  persona.md
  procesy.md
```

## Krok 4: sprawdz kontekst

```plain text
Przeczytaj folder kontekst/ i powiedz mi, czy ten kontekst wystarczy, zeby przejsc do M2 i M3. Jesli czegos brakuje, zadaj maksymalnie 5 pytan.
```

## Efekt koncowy M1

Claude Code zna Twoj zawodowy i osobisty kontekst. Od teraz kolejne skille nie zgaduja od zera.
